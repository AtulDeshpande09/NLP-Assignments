# NLP-Assignments

A comprehensive and structured learning repository dedicated to mastering Natural Language Processing (NLP). 

## Overview

This repository was created with a singular focus: to build a deep, intuitive understanding of NLP concepts. Rather than relying solely on high-level libraries, a significant portion of the foundational algorithms in this repository are implemented entirely from scratch. This approach ensures a thorough grasp of the underlying mathematics, logic, and mechanics of text processing.

To maintain a clear and structured learning path, the repository is organized into progressive assignments. Additionally, it includes a collection of rapid, one-hour applied projects. These quick projects are designed to provide practical, end-to-end experience with modern tools without diverting significant time away from the core theoretical learning process.

## Repository Structure

The repository is divided into three primary directories, each serving a specific educational purpose:

### 1. Assignments (`/assignments`)
A highly structured, progressive curriculum that covers the modern NLP pipeline from start to finish. The assignments are designed to build upon one another, starting with basic data gathering and culminating in advanced generative AI. 
* **Progression:** Web scraping and text preprocessing -> Classical NLP (POS tagging, NER, Topic Modeling, Vectorization) -> Deep Learning (Fine-tuning BERT and GPT-2).
* **Focus:** Applying industry-standard libraries (spaCy, Scikit-learn, Hugging Face) to solve specific, well-defined problems.

### 2. Others (`/others`)
The foundational core of the repository. This directory contains from-scratch implementations of fundamental NLP algorithms and text processing utilities.
* **Contents:** Custom implementations of Bag of Words, TF-IDF, Latent Dirichlet Allocation (LDA), N-grams, Skip-gram word embeddings, and various tokenization techniques (including BPE). It also includes a Context-Free Grammar (CFG) parser and a basic Named Entity Recognition (NER) model.
* **Focus:** Understanding the raw mechanics of NLP without the abstraction of high-level frameworks.

### 3. Projects (`/projects`)
A collection of applied, end-to-end implementations. 
* **Contents:** Includes a full-stack Seq2Seq agricultural chatbot (PyTorch, Flask, React), a Selenium-based YouTube automation script, and a biological sequence embedding model inspired by Word2Vec.
* **Focus:** Rapid, one-hour implementations designed to demonstrate practical application and full-stack integration without becoming overly time-consuming.

## Technologies and Frameworks

While the core focus is on the underlying concepts, the implementations utilize a wide range of modern data science and machine learning tools, including:
* **Core Languages:** Python
* **Deep Learning & NLP:** PyTorch, Hugging Face Transformers, spaCy, NLTK, Gensim
* **Machine Learning:** Scikit-learn, NumPy, Pandas
* **Web & Automation:** Flask, React, TailwindCSS, Selenium, BeautifulSoup
